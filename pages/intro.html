เริ่มต้นใช้งานการพัฒนา Add-Ons สําหรับ Bedrock Edition

ไม่ว่าคุณจะสร้างมันขึ้นมาเองหรือได้รับจากผู้สร้างคนอื่น Add-Ons เป็นขั้นตอนแรกในการเดินทางของการนําการปรับแต่งในระดับที่สูงขึ้นมาสู่ Minecraft: Bedrock Edition ส่วนเสริมช่วยให้ผู้เล่นสามารถเปลี่ยนรูปลักษณ์ของโลกของพวกเขาและแม้กระทั่งเปลี่ยนพฤติกรรมของเอนทิตี ตัวอย่างเช่น คุณสามารถเปลี่ยนรัศมีการระเบิดของไม้เลื้อยและพื้นผิวที่มันสวมใส่ได้

Add-On ประกอบด้วย:

  • Behavior Pack (BP)
  • Resource Pack (RP)
  • Script API (JavaScript)
