เริ่มต้นใช้งานการพัฒนา Add-Ons
สำหรับ Bedrock Edition

ไม่ว่าคุณจะสร้างขึ้นมาเองหรือได้รับจากผู้สร้างคนอื่น Add-Ons คือก้าวแรกในการนำการปรับแต่งระดับสูงมาสู่ Minecraft: Bedrock Edition ส่วนเสริมช่วยให้ผู้เล่นสามารถเปลี่ยนรูปลักษณ์ของโลก และแม้กระทั่งเปลี่ยนพฤติกรรมของเอนทิตีได้

Add-On ประกอบด้วย:

  • Behavior Pack (BP)
    • Script API
  • Resource Pack (RP)
